# PRD – BranX Premium Skincare Website (Awwwards Level)

## Project Goal

Build an ultra-premium, luxury skincare website for **BranX** using **Next.js**. The final result must look like it was designed by an award-winning UI/UX agency. Every section should feel cinematic with smooth animations, premium interactions, and a luxury skincare aesthetic.

# Tech Stack (Strict)

Use only production-ready technologies.

* Next.js (Latest App Router)
* TypeScript
* Tailwind CSS
* GSAP
* Framer Motion
* Lenis Smooth Scroll
* SplitType
* React Icons
* Lucide Icons
* shadcn/ui
* Aceternity UI
* Magic UI
* Motion Primitives
* React Intersection Observer
* next/image
* next/font
* ESLint
* Prettier

Initialize premium UI components.

```bash
npm install gsap
npm install framer-motion
npm install @studio-freight/lenis
npm install split-type
npm install lucide-react
npm install react-icons

npx shadcn@latest init

uipro init --ai claude
```

Everything must be production-ready.

No warnings.

No hydration issues.

No console errors.

No layout shifts.

Deploy directly on Vercel without changing any code.
